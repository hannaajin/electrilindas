¿Cómo funciona el guardado?

Al hacer clic en Guardar cambios, este panel commitea content.json directamente a tu repositorio de GitHub via API. Netlify detecta el commit y redeploya el sitio en ~30 segundos.

  1. El repositorio ya está configurado: hannaajin/electrilindas
  2. Ingresá tu Personal Access Token una vez (se guarda en tu navegador)
  3. Desde ahí, cada "Guardar cambios" actualiza el sitio automáticamente

🔑 Token de GitHub

⚠️ No compartas este token. Se guarda solo en tu navegador (localStorage) y nunca se sube al repositorio. Si alguna vez lo compartís, andá a github.com/settings/tokens y eliminalo para crear uno nuevo.
Necesitás un token con permiso repo. Creá uno en: github.com/settings/tokens/new
